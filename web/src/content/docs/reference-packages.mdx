## @x402x/core

Core utilities and network configuration.

- Network helpers: `getNetworkConfig(network)`, `getSupportedNetworks()`
- Settlement helpers: `addSettlementExtra(requirements, { hook, hookData, facilitatorFee?, payTo?, salt? })`
- Commitment: `calculateCommitment(params)`, `generateSalt()`
- Built‑in hooks: `TransferHook.getAddress(network)`, `TransferHook.encode(splits?)`
- Facilitator helpers: `isSettlementMode(req)`, `validateSettlementRouter(...)`,
  `settleWithRouter(signer, payload, requirements, config)`, `calculateFacilitatorFee(url, network, hook, hookData?)`

Example:

```core-example.ts
import { addSettlementExtra, TransferHook, calculateCommitment } from "@x402x/core";
```

## @x402x/fetch

Fetch wrapper that completes 402 flows automatically and supports settlement mode.

- `wrapFetchWithPayment(fetch, walletClient, maxValue?, selector?, config?)`
- Also exports `decodeXPaymentResponse`, `createSigner` from x402

Example:

```fetch-example.ts
const fetchWithPay = wrapFetchWithPayment(fetch, walletClient);
const res = await fetchWithPay("/api/protected");
```

## @x402x/hono

Hono middleware for protecting routes and settling via facilitator.

- `paymentMiddleware(payTo, routes, facilitatorConfig?)`
- Adds `c.get("x402")` context: payer, amount, network, payment, settlement?

```hono-middleware.ts
app.use("/api/*", paymentMiddleware("0xRecipient...", { price: "$0.10", network: "base-sepolia" }, { url }));
```

## @x402x/express

Express middleware mirroring the Hono API.

- `paymentMiddleware(payTo, routes, facilitatorConfig?)`
- Adds `req.x402`

```express-middleware.ts
app.use(paymentMiddleware("0xRecipient...", { price: "$0.10", network: "base-sepolia" }, { url }));
```

## @x402x/react

React helpers for client apps.

- `useX402Payment({ maxValue? })` → `pay(url)`, `status`, `error`, `result`

```react-example.ts
const { pay, status } = useX402Payment();
await pay("/api/premium");
```

## @x402x/client

Serverless client SDK for talking to the facilitator directly.

- Class: `new X402Client({ wallet, network, facilitatorUrl, ... })`
- Methods: `execute({...})`
- Primitives: `prepareSettlement`, `signAuthorization`, `submitToFacilitator`

```client-example.ts
const client = new X402Client({ wallet, network: "base-sepolia", facilitatorUrl: url });
await client.execute({ hook, hookData, amount, recipient });
```

## Facilitator Service

A production‑ready facilitator lives under `facilitator/`.

- Dual‑mode (standard x402 and settlement)
- Security: CORS, rate‑limit, hook whitelist, gas caps, fee validation
- Observability: metrics + OpenTelemetry
- Endpoints: `/verify`, `/settle`, `/calculate-fee` (aka `min-facilitator-fee`), `/health`, `/ready`

See `facilitator/README.md` for deployment and configuration.
